# Books API

Simple express API built using Node/Express, MongoDB cluster for persistence, OpenApi 3.0 for request and response validation.

## Setup

```bash
git clone git@github.com:nkhil/books-api.git
cd books-api
npm install
npm run develop
# The app should run on port 8080
```

Note: You will need to supply the following credentials for a MongoDB cluster as environment variables.

- Database name
- Password
